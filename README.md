# CarbonStep – Aplicación para medir la huella de carbono personal

CarbonStep es un **prototipo funcional** de una aplicación web cuyo objetivo es ayudar a los usuarios a medir, entender y reducir su huella de carbono personal.  
La aplicación permite calcular emisiones en distintas categorías, visualizar el progreso, participar en retos, formar parte de comunidades y ver el impacto colectivo.

Este proyecto se ha desarrollado como parte de una práctica sobre sostenibilidad, aplicando conceptos tecnológicos y sociales relacionados con la reducción de emisiones de CO₂.

---

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS
- HTML

No se utiliza backend real ni bases de datos externas. Los datos se gestionan en memoria como parte del prototipo.

---

## Cómo ejecutar la aplicación

### Requisitos previos

- Tener instalado **Node.js** y **npm**

### Pasos para ejecutar el proyecto

1. Clonar el repositorio o descargar el proyecto.

2. Instalar las dependencias (desde la carpeta del proyecto):

   ```bash
   npm install
   ```

3. Ejecutar la aplicación en modo desarrollo:

   ```bash
   npm run dev
   ```

4. Abrir en el navegador la URL:
   http://localhost:5173/

## Prototipo funcional

La aplicación representa un MVP (Minimum Viable Product) cuyo objetivo es demostrar el funcionamiento principal del sistema:

- Cálculo real de la huella de carbono.
- Visualización del progreso personal y colectivo.
- Simulación de recomendaciones, gamificación y comunidad.

Las funcionalidades más complejas (APIs externas, machine learning avanzado o bases de datos reales) están simuladas, pero la lógica de cálculo y la experiencia de usuario están implementadas.

## Plan de lanzamiento y crecimiento (marketing digital sostenible)

Para el lanzamiento de la aplicación CarbonStep se propone una estrategia de crecimiento progresiva y sostenible, centrada en concienciar a los usuarios y evitar técnicas de marketing agresivas.

### Fase 1: Lanzamiento beta

- Duración aproximada: 1–2 meses.
- Público objetivo: estudiantes y personas interesadas en sostenibilidad.
- Difusión a través de centros educativos y comunidades locales.
- Recogida de feedback para mejorar la experiencia de usuario y ajustar funcionalidades.

### Fase 2: Lanzamiento oficial

- Publicación de la aplicación con las funcionalidades principales.
- Campañas en redes sociales (Instagram, TikTok y X) con contenido educativo sobre huella de carbono.
- Colaboración con microinfluencers relacionados con sostenibilidad y tecnología.
- Retos iniciales dentro de la aplicación para fomentar la participación desde el primer uso.

### Fase 3: Crecimiento y fidelización

- Creación de comunidades locales dentro de la aplicación.
- Retos colectivos mensuales con recompensas sostenibles.
- Colaboraciones con comercios y servicios sostenibles para ofrecer descuentos a los usuarios activos.
- Envío de notificaciones responsables, solo cuando aporten valor al usuario.

### Enfoque de marketing sostenible

- Comunicación clara y transparente, evitando el greenwashing.
- Prioridad al contenido educativo frente a la publicidad agresiva.
- Respeto a la privacidad del usuario y uso responsable de los datos.
- Promoción del impacto colectivo y de los logros reales alcanzados por la comunidad.
